# Direttive

Questa cartella contiene le SOP (Standard Operating Procedures) scritte in Markdown.

Ogni direttiva definisce:
- **Obiettivo**: Cosa deve essere fatto
- **Input**: Dati o parametri necessari
- **Tool/Script**: Quali script in `execution/` usare
- **Output**: Il risultato atteso
- **Casi limite**: Situazioni particolari da gestire
