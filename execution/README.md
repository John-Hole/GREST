# Execution

Questa cartella contiene gli script Python deterministici che eseguono il lavoro effettivo.

Principi:
- **Deterministici**: Stesso input → stesso output
- **Ben commentati**: Ogni script documenta cosa fa, i parametri e il formato di output
- **Testabili**: Ogni script può essere eseguito indipendentemente
- **Veloci**: Ottimizzati per performance

Le variabili d'ambiente e i token API sono nel file `.env` nella root del progetto.
